# NeuroScreen: Intelligent NLP-Powered Resume Classification Engine

NeuroScreen is an automated, end-to-end intelligent applicant screening system designed to streamline high-volume talent acquisition pipelines. By leveraging Natural Language Processing and supervised machine learning, this engine strips formatting noise from raw resumes, extracts high-dimensional technical keywords, and automatically categorizes applicants into 20+ professional job domains with an exceptional 97% classification accuracy.

The system features an automated text sanitization pipeline that uses custom regular expression wrappers to filter out structural noise, URLs, hashtags, special characters, and linguistic stopwords. For feature engineering, it implements TF-IDF Vectorization to evaluate the mathematical weight of specific technical terms rather than relying on crude token counts. This semantic data is then processed through a high-dimensional multi-class modeling framework that pairs a distance-based K-Nearest Neighbors classifier with a One-Vs-Rest wrapper to handle complex, multi-label domain separation. Finally, the architecture utilizes binary serialization via Pickle to decouple training from deployment, enabling lightweight, sub-millisecond production inference on standard hardware.

The underlying infrastructure relies on a highly efficient, data-centric stack. The core machine learning algorithms and math operations are built entirely on Scikit-Learn, utilizing its TfidfVectorizer, OneVsRestClassifier, and KNeighborsClassifier packages. Dedicated natural language operations are managed through the Natural Language Toolkit (NLTK) English stopwords corpus, alongside Python's native regular expressions engine (`re`) for deterministic character stripping. Data transformation, table structures, and matrix manipulations are optimized using the foundational Pandas and NumPy ecosystem, which handles the large-scale multidimensional sparse arrays before they are ingested by the model.

To run the project locally, users can clone the repository from GitHub, install the necessary dependencies from the requirements file using pip, and execute the core application script to launch the automated data flow pipeline.
